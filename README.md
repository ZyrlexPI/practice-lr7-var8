Данный репозиторий содержит лабораторную работу №7 вариант 8 (7+1).

**Задание:** Дана  матрица  А(5?5).  Определить  массив  Х  из  5  элементов, 
каждый  из  которых  равен  1,  если  в  соответствующей  строке 
элемент  главной  диагонали  больше  элемента  побочной  и  –1  в 
противном  случае.  Определить  величину  Y,  как  количество 
нечетных элементов в первом столбце матрицы A.
